---
title: Overview
description: Build JSON:API compliant responses
---

The `Response` class is the entry point for building all API responses. It provides two methods:

- `success()` for successful responses with data
- `error()` for error responses

```php
use Eufaturo\ApiToolkit\Http\Response;

final class ProductController
{
    public function __construct(
        private readonly Response $response,
    ) {
    }
}
```

## Inject via constructor

The `Response` class is registered as a singleton in the service container. Inject it via constructor dependency injection:

```php
public function __construct(
    protected readonly Response $response,
) {
}
```

## Content type

All responses automatically set the `Content-Type` header to `application/vnd.api+json` as required by the JSON:API specification.
