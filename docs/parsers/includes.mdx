---
title: Includes
description: Parse and apply include query parameters for eager loading
---

The `IncludeParser` handles the `?include` query parameter following the [JSON:API inclusion specification](https://jsonapi.org/format/#fetching-includes).

## Format

```
?include=category
?include=category,supplier
```

Comma-separated list of relationship names to eager-load.

## Usage

```php
use BlueBeetle\ApiToolkit\Parsers\IncludeParser;

$parser = new IncludeParser(allowed: ['category', 'supplier', 'tags']);

// Parses the request and eager-loads matching relationships
$parser->apply($request, $query);
```

## Parse without applying

To get the parsed list without applying to a query:

```php
$includes = $parser->parse($request);
// ['category', 'supplier']
```

## Allowed list

Only relationships in the `allowed` list are included:

```php
$parser = new IncludeParser(allowed: ['category']);

// ?include=category,secret → only 'category' is loaded
```

## Unrestricted mode

Pass an empty `allowed` array to allow all includes:

```php
$parser = new IncludeParser();

// ?include=anything,goes → both are loaded
```

<Warning>
  Unrestricted mode can expose internal relationships. Always define an explicit `allowed` list in production.
</Warning>
