---
title: Filters
description: Parse and apply filter query parameters
---

The `FilterParser` handles `?filter[field]=value` query parameters and applies them to an Eloquent query builder.

## Usage

```php
use BlueBeetle\ApiToolkit\Parsers\FilterParser;
use BlueBeetle\ApiToolkit\Parsers\Filters\ExactFilter;
use BlueBeetle\ApiToolkit\Parsers\Filters\PartialFilter;

$parser = new FilterParser([
    'status' => new ExactFilter(),
    'name' => new PartialFilter(),
]);

$parser->apply($request, $query);
```

## Built-in filter types

### ExactFilter

Matches the exact value. Supports arrays for `WHERE IN`:

```
?filter[status]=active          → WHERE status = 'active'
?filter[status][]=a&filter[status][]=b  → WHERE status IN ('a', 'b')
```

```php
'status' => new ExactFilter()
```

### PartialFilter

Performs a `LIKE` search:

```
?filter[name]=wid               → WHERE name LIKE '%wid%'
```

```php
'name' => new PartialFilter()
```

### DateFilter

Filters by date or date range:

```
?filter[created_at]=2025-01-15              → WHERE DATE(created_at) = '2025-01-15'
?filter[created_at][from]=2025-01-01        → WHERE DATE(created_at) >= '2025-01-01'
?filter[created_at][to]=2025-12-31          → WHERE DATE(created_at) <= '2025-12-31'
?filter[created_at][from]=2025-01-01&filter[created_at][to]=2025-12-31  → both
```

```php
'created_at' => new DateFilter()
```

### ScopeFilter

Delegates to an Eloquent scope on the model:

```
?filter[active]=yes             → $query->active('yes')
```

```php
// Uses the field name as the scope
'active' => new ScopeFilter()

// Or specify a custom scope name
'status' => new ScopeFilter(scopeName: 'whereStatus')
```

## Custom filters

Implement the `Filter` interface to create your own:

```php
use BlueBeetle\ApiToolkit\Parsers\Filters\Filter;
use Illuminate\Database\Eloquent\Builder;

final readonly class JsonContainsFilter implements Filter
{
    public function apply(Builder $query, string $field, mixed $value): void
    {
        $query->whereJsonContains($field, $value);
    }
}
```

```php
'tags' => new JsonContainsFilter()
```

## Filter resolution

Filters can be passed as instances or class strings:

```php
// Instance
'status' => new ExactFilter()

// Class string (instantiated when needed)
'status' => ExactFilter::class
```

## Unknown filters

Filters not listed in the `allowed` array are silently ignored. They are never applied to the query.
