## ValidPageSize

Validates that the page size is one of the allowed values:

```php
use BlueBeetle\ApiToolkit\Rules\ValidPageSize;

public function queryParamRules(): array
{
    return [
        'page.size' => ['numeric', new ValidPageSize()],
    ];
}
```

### Default valid sizes

`10, 20, 40, 80, 100`

### Custom valid sizes

```php
new ValidPageSize([25, 50, 75, 100])
```

### Error message

```
The [15] is not a valid page size value.
```

## ValidInclude

Validates that the requested includes are in the allowed list:

```php
use BlueBeetle\ApiToolkit\Rules\ValidInclude;

public function queryParamRules(): array
{
    return [
        'include' => [new ValidInclude(['category', 'supplier', 'tags'])],
    ];
}
```

### Accepts

- Comma-separated string: `category,supplier`
- Array: `['category', 'supplier']`

### Error message

```
The [unknown,invalid] is not a valid include value.
```
