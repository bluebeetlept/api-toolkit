## ForceJsonApiResponse

Sets the `Content-Type` header to `application/vnd.api+json` on all responses:

```php
use BlueBeetle\ApiToolkit\Http\Middleware\ForceJsonApiResponse;
```

Register in your middleware stack:

```php bootstrap/app.php
->withMiddleware(function (Middleware $middleware) {
    $middleware->api(prepend: [
        ForceJsonApiResponse::class,
    ]);
})
```

## DetectLanguage

Reads the `Language` header from the request and sets the application locale:

```php
use BlueBeetle\ApiToolkit\Http\Middleware\DetectLanguage;
```

### Configuration

The middleware reads from your `config/app.php`:

```php config/app.php
'valid_locales' => ['en', 'pt', 'es'],
'fallback_locale' => 'en',
```

### Behavior

| `Language` header | `valid_locales` | Result |
|-------------------|-----------------|--------|
| `pt` | `['en', 'pt']` | Locale set to `pt` |
| `fr` | `['en', 'pt']` | Locale set to `en` (fallback) |
| _(not set)_ | - | No change |

The header name is case-insensitive and the value is lowercased before matching.
