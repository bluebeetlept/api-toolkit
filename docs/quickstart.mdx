This guide walks you through creating a complete JSON:API endpoint for a `Product` model.

## 1. Define a Resource

Create a resource that describes how your model is serialized:

```php app/Http/Resources/ProductResource.php
use BlueBeetle\ApiToolkit\Parsers\Filters\ExactFilter;
use BlueBeetle\ApiToolkit\Parsers\Filters\PartialFilter;
use BlueBeetle\ApiToolkit\Resources\Resource;
use App\Models\Product;

final class ProductResource extends Resource
{
    protected string $model = Product::class;

    public function attributes(Product $product): array
    {
        return [
            'name' => $product->name,
            'code' => $product->code,
            'description' => $product->description,
        ];
    }

    public function self(Product $product): string | null
    {
        return route('api.v1.products.show', $product);
    }

    public function relationships(): array
    {
        return [
            'category' => CategoryResource::class,
        ];
    }

    public function allowedFilters(): array
    {
        return [
            'name' => new PartialFilter(),
            'code' => new ExactFilter(),
        ];
    }

    public function allowedSorts(): array
    {
        return ['name', 'created_at'];
    }

    public function defaultSort(): string | null
    {
        return '-created_at';
    }
}
```

## 2. Create a list endpoint

```php app/Http/Controllers/Products/ListController.php
use App\Http\Resources\ProductResource;
use App\Models\Product;
use BlueBeetle\ApiToolkit\QueryBuilder;
use Illuminate\Http\Request;

final class ListController
{
    public function __invoke(Request $request)
    {
        return QueryBuilder::for(Product::class, $request)
            ->fromResource(ProductResource::class)
            ->paginate();
    }
}
```

## 3. Create a view endpoint

```php app/Http/Controllers/Products/ViewController.php
use App\Http\Resources\ProductResource;
use App\Models\Product;
use BlueBeetle\ApiToolkit\Http\Response;

final class ViewController
{
    public function __invoke(Product $product, Response $response)
    {
        return $response->success($product, ProductResource::class);
    }
}
```

## 4. Try it out

```bash
# List with filtering and sorting
curl "https://api.example.com/products?filter[name]=widget&sort=-created_at&page[size]=20"

# View a single product with relationships
curl "https://api.example.com/products/abc-123?include=category"
```

### Single resource response

```json
{
  "data": {
    "type": "products",
    "id": "abc-123",
    "attributes": {
      "name": "Widget",
      "code": "W01",
      "description": "A fine widget"
    },
    "relationships": {
      "category": {
        "data": { "type": "categories", "id": "5" }
      }
    },
    "links": {},
    "meta": {}
  },
  "included": [
    {
      "type": "categories",
      "id": "5",
      "attributes": { "name": "Gadgets" }
    }
  ]
}
```
